## 1. ログイン

### 1.1 ログイン手順

1. システムのURLにアクセスします
2. 「Larkでログイン」ボタンをクリックします
3. Larkのログイン画面が表示されます
4. 社内のLarkアカウントでログインします
5. ログイン成功後、ダッシュボードが表示されます

### 1.2 ログアウト

画面右上のユーザーアイコンをクリックし、「ログアウト」を選択します。

---

## 2. ダッシュボード

ダッシュボードでは、あなたの書類申請状況を一覧で確認できます。

### 2.1 画面構成

| カード | 説明 |
|--------|------|
| 免許証 | 運転免許証の申請状況（1件のみ） |
| 車検証 | 車検証の申請状況（複数可能） |
| 任意保険証 | 任意保険証の申請状況（複数可能） |

### 2.2 ステータス表示

| ステータス | 意味 |
|------------|------|
| 未申請 | まだ申請していません |
| 審査中 | 申請済み、承認待ちです |
| 承認済み | 承認されました |
| 却下 | 申請が却下されました（再申請が必要） |

### 2.3 新規申請

各カードをクリックすると、申請画面が開きます。

## 3. 運転免許証の申請

### 3.1 申請手順

1. ダッシュボードで「免許証」カードをクリック
2. 申請フォームに以下の情報を入力：

| 項目 | 入力内容 | 必須 |
|------|----------|------|
| 免許証番号 | 12桁の免許証番号 | ○ |
| 免許種別 | 普通/中型/大型など | ○ |
| 交付日 | 免許証の交付日 | - |
| 有効期限 | 免許証の有効期限 | ○ |
| 免許証画像 | 免許証の写真 | ○ |

3. 免許証の画像をアップロード
   - 「ファイルを選択」ボタンをクリック
   - 免許証の表面を撮影した画像を選択
   - 対応形式: JPG, PNG, PDF
   - 最大サイズ: 10MB

4. 「申請する」ボタンをクリック

### 3.2 注意事項

- 免許証は1人1件のみ登録可能です
- 更新する場合は、管理者に削除を依頼してから再申請してください
- 画像は鮮明に写っている必要があります

## 4. 車検証の申請

### 4.1 申請手順

1. ダッシュボードで「車検証」カードの「追加」ボタンをクリック
2. 申請フォームに以下の情報を入力：

| 項目 | 入力内容 | 必須 |
|------|----------|------|
| 車両番号 | ナンバープレート（例: 佐賀 500 あ 1234） | ○ |
| 車検有効期限 | 車検の有効期限 | ○ |
| 車検証画像 | 車検証の写真 | ○ |

3. 車検証の画像をアップロード
4. 「申請する」ボタンをクリック

### 4.2 複数車両の登録

- 複数の車両を登録できます
- 各車両ごとに個別に申請してください

## 5. 任意保険証の申請

### 5.1 申請手順

1. ダッシュボードで「任意保険証」カードの「追加」ボタンをクリック
2. 申請フォームに以下の情報を入力：

| 項目 | 入力内容 | 必須 |
|------|----------|------|
| 証券番号 | 保険証券の番号 | ○ |
| 保険会社 | 保険会社名 | ○ |
| 保険種別 | 対人・対物・車両など | ○ |
| 保障開始日 | 保険の開始日 | ○ |
| 保障終了日 | 保険の終了日 | ○ |
| 対人賠償 | 無制限 or 金額 | ○ |
| 対物賠償 | 賠償金額 | ○ |
| 搭乗者傷害 | 傷害金額 | - |
| 保険証画像 | 保険証券の写真 | ○ |

3. 保険証券の画像をアップロード
4. 「申請する」ボタンをクリック

## 6. マイ書類の確認

### 6.1 書類一覧の表示

1. サイドメニューから「マイ書類」をクリック
2. 登録済みの全書類が一覧表示されます

### 6.2 書類詳細の確認

- 各書類をクリックすると詳細が表示されます
- アップロードした画像も確認できます

### 6.3 却下された場合

却下された書類には却下理由が表示されます。内容を確認し、修正して再申請してください。

---

## 7. 許可証の確認・印刷

### 7.1 許可証とは

全ての書類（免許証、車検証、任意保険証）が承認されると、「通勤車両使用許可証」が発行されます。

### 7.2 許可証の確認手順

1. サイドメニューから「許可証」をクリック
2. 発行済みの許可証一覧が表示されます
3. 確認したい許可証をクリック

### 7.3 許可証の印刷

1. 許可証詳細画面で「印刷」ボタンをクリック
2. PDFがダウンロードされます
3. PDFを開いて印刷してください

### 7.4 許可証のQRコード

許可証にはQRコードが印刷されています。このQRコードをスマートフォンで読み取ると、許可証の有効性を確認できます。

## よくある質問（FAQ）

### Q1. 申請後、どのくらいで承認されますか？
A1. 通常、1〜3営業日以内に審査が行われます。

### Q2. 書類を修正したい場合はどうすればいいですか？
A2. 管理者に連絡して、書類の削除を依頼してください。削除後、再申請できます。

### Q3. 有効期限が近づいたらどうなりますか？
A3. 有効期限の設定日数前（通常30日前）にLarkで通知が届きます。早めに更新手続きをしてください。

### Q4. 画像がアップロードできません
A4. 以下を確認してください：
- ファイル形式: JPG, PNG, PDF
- ファイルサイズ: 10MB以下
- 画像が破損していないこと
