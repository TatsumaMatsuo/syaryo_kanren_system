## 2. 申請一覧・承認

### 2.1 申請一覧画面

申請一覧画面では、全社員からの申請を確認できます。

#### フィルター機能

| フィルター | 説明 |
|------------|------|
| 審査待ち | 承認待ちの申請のみ表示 |
| 承認済み | 承認された申請のみ表示 |
| 却下 | 却下された申請のみ表示 |
| 全て | 全ての申請を表示 |

#### 表示項目

- 申請者名
- 書類種別（免許証/車検証/任意保険証）
- 申請日時
- ステータス

### 2.2 申請の承認（管理者のみ）

1. 申請一覧から承認する申請をクリック
2. 書類詳細画面が開きます
3. アップロードされた画像を確認
4. 入力内容を確認
5. 以下のいずれかを選択：
   - **承認**: 「承認」ボタンをクリック
   - **却下**: 「却下」ボタンをクリックし、却下理由を入力

### 2.3 承認時の注意事項

- 画像が鮮明であることを確認
- 入力内容と画像の内容が一致していることを確認
- 有効期限が正しく入力されていることを確認

## 3. 社員別書類確認

### 3.1 社員一覧画面

社員一覧画面では、社員ごとの書類登録状況を確認できます。

#### 表示項目

- 社員名
- 社員ID
- 免許証: ○/×（登録済み/未登録）
- 車検証: 登録件数
- 任意保険証: 登録件数
- 全書類承認状況

### 3.2 社員詳細の確認

1. 社員一覧から確認したい社員をクリック
2. その社員の全書類が表示されます
3. 各書類をクリックすると詳細が確認できます

## 4. 代理申請

### 4.1 代理申請とは

社員本人に代わって、管理者・閲覧者が書類申請を行う機能です。

### 4.2 代理申請の手順

1. サイドメニューから「代理申請」をクリック
2. 社員検索画面が表示されます
3. 社員名またはIDで検索
4. 対象社員を選択
5. 申請したい書類の「追加」ボタンをクリック
6. 申請フォームに必要情報を入力
7. 「申請する」ボタンをクリック

### 4.3 注意事項

- 代理申請した書類も通常の承認フローが適用されます
- 代理申請であることは記録に残ります

### 5.1 検索機能

## 6. 許可証発行

### 6.1 許可証発行条件

以下の条件を全て満たす社員に許可証が発行されます：

- 運転免許証が承認済み
- 車検証が1件以上承認済み
- 任意保険証が1件以上承認済み

### 6.2 許可証一覧画面

許可証一覧画面では、発行済み許可証を確認できます。

#### 表示項目

- 社員名
- 発行日
- 有効期限（最も早く期限切れになる書類の期限）
- ステータス

### 6.3 許可証の発行・再発行

1. 許可証一覧画面で「発行」ボタンをクリック
2. PDFがダウンロードされます

### 6.4 許可証の内容

許可証には以下が記載されます：

- 社員名・社員ID
- 免許証番号・有効期限
- 車両番号・車検有効期限
- 保険証券番号・保障期間
- 発行日・有効期限
- QRコード（検証用）

## 7. 有効期限監視

### 7.1 監視画面

有効期限監視画面では、期限切れ間近・期限切れの書類を確認できます。

### 7.2 表示内容

| セクション | 説明 |
|------------|------|
| 期限切れ間近 | 設定日数以内に期限が切れる書類 |
| 期限切れ | 既に期限が切れている書類 |

### 7.3 警告設定（システム設定より）

各書類タイプごとに警告日数が設定されています：

- 免許証: X日前から警告
- 車検証: X日前から警告
- 任意保険証: X日前から警告
- 管理者エスカレーション: 期限切れ後X日で管理者に通知

### 7.4 手動実行（管理者のみ）

「手動実行」ボタンをクリックすると、有効期限チェックと通知送信を即座に実行できます。

## 9. 設定（管理者のみ）

### 9.1 権限管理

#### ユーザー権限の付与

1. サイドメニューから「設定」→「権限管理」をクリック
2. 「権限を追加」ボタンをクリック
3. Larkユーザーを検索して選択
4. 権限（管理者/閲覧者）を選択
5. 「追加」ボタンをクリック

#### ユーザー権限の変更

1. 権限一覧から対象ユーザーを選択
2. 権限を変更
3. 「保存」ボタンをクリック

#### ユーザー権限の削除

1. 権限一覧から対象ユーザーの「削除」ボタンをクリック
2. 確認ダイアログで「削除」をクリック

### 9.2 システム設定

#### 会社情報（許可証PDF用）

| 設定項目 | 説明 |
|----------|------|
| 会社名 | 許可証に印字される会社名 |
| 郵便番号 | 会社の郵便番号 |
| 住所 | 会社の住所 |
| 発行部署 | 許可証の発行部署名 |

#### 有効期限警告設定

| 設定項目 | 説明 | デフォルト |
|----------|------|------------|
| 免許証警告日数 | 有効期限の何日前に警告するか | 30日 |
| 車検証警告日数 | 車検有効期限の何日前に警告するか | 30日 |
| 任意保険証警告日数 | 保障終了日の何日前に警告するか | 30日 |
| 管理者エスカレーション日数 | 期限切れ後何日で管理者に通知するか | 7日 |

## トラブルシューティング

### Q1. 承認ボタンが表示されない
A1. 閲覧者権限では承認できません。管理者に連絡してください。

### Q2. 社員が検索で見つからない
A2. Larkに登録されている社員のみ検索可能です。人事部に確認してください。

### Q3. 許可証が発行できない
A3. 以下を確認してください：
- 免許証が承認済みか
- 車検証が1件以上承認済みか
- 任意保険証が1件以上承認済みか

### Q4. 通知が届かない
A4. 以下を確認してください：
- 対象社員のLark IDが正しいか
- Larkの通知設定が有効か
- システム設定の警告日数が適切か
