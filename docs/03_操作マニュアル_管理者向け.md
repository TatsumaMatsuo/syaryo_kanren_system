## 2. 申請一覧・承認

### 2.1 申請一覧画面

申請一覧画面では、全社員からの申請を確認できます。

#### フィルター機能

| フィルター | 説明 |
|------------|------|
| 審査待ち | 承認待ちの申請のみ表示 |
| 承認済み | 承認された申請のみ表示 |
| 却下 | 却下された申請のみ表示 |
| 全て | 全ての申請を表示 |

#### 表示項目

- 申請者名
- 書類種別（免許証/車検証/任意保険証）
- 申請日時
- ステータス

### 2.2 申請の承認（管理者のみ）

1. 申請一覧から承認する申請をクリック
2. 書類詳細画面が開きます
3. アップロードされた画像を確認
4. 入力内容を確認
5. 以下のいずれかを選択：
   - **承認**: 「承認」ボタンをクリック
   - **却下**: 「却下」ボタンをクリックし、却下理由を入力

### 2.3 承認時の注意事項

- 画像が鮮明であることを確認
- 入力内容と画像の内容が一致していることを確認
- 有効期限が正しく入力されていることを確認
- **免許証の場合**: 表面・裏面の画像が正しく横向きで撮影されていることを確認

## 3. 一括承認

### 3.1 一括承認とは

複数の申請をまとめて承認・却下できる機能です（管理者のみ）。

### 3.2 一括承認の手順

1. 申請一覧画面で「一括承認モード」をオンにする
2. 承認・却下したい申請のチェックボックスを選択
3. 画面下部に選択数が表示されます
4. 以下のいずれかを選択：
   - **一括承認**: 「一括承認」ボタンをクリック
   - **一括却下**: 「一括却下」ボタンをクリックし、却下理由を入力

### 3.3 注意事項

- 一度に承認できる申請数に制限はありません
- 一括却下の場合、全ての申請に同じ却下理由が適用されます
- 承認・却下後は承認履歴に記録されます

## 4. 社員別書類確認

### 4.1 社員一覧画面

社員一覧画面では、社員ごとの書類登録状況を確認できます。

#### 表示項目

- 社員名
- 社員ID
- 免許証: ○/×（登録済み/未登録）
- 車検証: 登録件数
- 任意保険証: 登録件数
- 全書類承認状況

### 4.2 社員詳細の確認

1. 社員一覧から確認したい社員をクリック
2. その社員の全書類が表示されます
3. 各書類をクリックすると詳細が確認できます

### 4.3 免許証画像の表示

免許証は表面・裏面の2つの画像が縦2段で表示されます。
- **表面**: 免許証の表面画像（必須）
- **裏面**: 免許証の裏面画像（任意）

各画像にはダウンロードボタンが付いており、個別にダウンロードできます。

## 5. 代理申請

### 5.1 代理申請とは

社員本人に代わって、管理者・閲覧者が書類申請を行う機能です。

### 5.2 代理申請の手順

1. サイドメニューから「代理申請」をクリック
2. 社員検索画面が表示されます
3. 社員名またはIDで検索
4. 対象社員を選択
5. 申請したい書類の「追加」ボタンをクリック
6. 申請フォームに必要情報を入力
7. 「申請する」ボタンをクリック

### 5.3 注意事項

- 代理申請した書類も通常の承認フローが適用されます
- 代理申請であることは記録に残ります

## 6. 社員検索

### 6.1 検索機能

社員名、社員ID、書類番号などで検索できます。

### 6.2 検索手順

1. サイドメニューから「社員検索」をクリック
2. 検索ボックスにキーワードを入力
3. 検索結果から対象を選択

### 6.3 書類画像の表示

検索結果では、各書類の詳細情報と画像が表示されます。

- **免許証**: 表面・裏面の画像が縦2段で表示されます
- **車検証**: 車検証画像が表示されます
- **任意保険証**: 保険証画像が表示されます

## 7. 許可証発行

### 7.1 許可証発行条件

以下の条件を全て満たす社員に許可証が発行されます：

- 運転免許証が承認済み
- 車検証が1件以上承認済み
- 任意保険証が1件以上承認済み

### 7.2 許可証一覧画面

許可証一覧画面では、発行済み許可証を確認できます。

#### 表示項目

- 社員名
- 発行日
- 有効期限（最も早く期限切れになる書類の期限）
- ステータス

### 7.3 許可証の発行・再発行

1. 許可証一覧画面で「発行」ボタンをクリック
2. PDFがダウンロードされます

### 7.4 許可証の内容

許可証には以下が記載されます：

- 社員名・社員ID
- 免許証番号・有効期限
- 車両番号・車検有効期限
- 保険証券番号・保障期間
- 発行日・有効期限
- QRコード（検証用）

## 8. 有効期限監視

### 8.1 監視画面

有効期限監視画面では、期限切れ間近・期限切れの書類を確認できます。

### 8.2 表示内容

| セクション | 説明 |
|------------|------|
| 期限切れ間近 | 設定日数以内に期限が切れる書類 |
| 期限切れ | 既に期限が切れている書類 |

### 8.3 警告設定（システム設定より）

各書類タイプごとに警告日数が設定されています：

- 免許証: X日前から警告
- 車検証: X日前から警告
- 任意保険証: X日前から警告
- 管理者エスカレーション: 期限切れ後X日で管理者に通知

### 8.4 手動実行（管理者のみ）

「手動実行」ボタンをクリックすると、有効期限チェックと通知送信を即座に実行できます。

## 9. 通知履歴

### 9.1 通知履歴画面

送信済みの通知を確認できます。

### 9.2 表示項目

## 10. 分析ダッシュボード

### 10.1 概要

分析ダッシュボードでは、システム全体の状況をグラフで可視化できます。

### 10.2 表示内容

| セクション | 説明 |
|------------|------|
| 申請状況 | 書類種別ごとの審査待ち・承認済み・却下の件数（円グラフ） |
| 期限切れ状況 | 期限切れ・期限間近の書類件数（棒グラフ） |
| 承認履歴 | 過去の承認・却下の推移（折れ線グラフ） |
| サマリー | 総申請数、承認率、期限切れ件数などの統計 |

### 10.3 フィルター

- 期間指定（過去7日、30日、90日など）
- 書類種別（免許証、車検証、任意保険証）

## 11. エクスポート

### 11.1 概要

登録データをCSVまたはExcel形式でダウンロードできます。

### 11.2 エクスポート対象

| データ種別 | 内容 |
|------------|------|
| 免許証 | 社員ID、社員名、免許番号、有効期限、承認状態など |
| 車検証 | 社員ID、社員名、車両番号、車検有効期限、承認状態など |
| 任意保険証 | 社員ID、社員名、証券番号、補償期間、承認状態など |
| 承認履歴 | 日時、社員名、書類種別、アクション、承認者名など |
| 全データ | 上記すべてを含む（Excel形式のみ、シート分け） |

### 11.3 エクスポート手順

1. サイドメニューから「エクスポート」をクリック
2. エクスポートするデータ種別を選択
3. 出力形式（CSV/Excel）を選択
4. 承認履歴の場合は期間を指定（任意）
5. 「エクスポート」ボタンをクリック
6. ファイルがダウンロードされます

### 11.4 注意事項

- CSV形式は1種類のデータのみ出力可能
- Excel形式は複数シートに分けて出力
- 日本語文字化け対策として、CSVはBOM付きUTF-8で出力

## 12. 設定（管理者のみ）

### 12.1 権限管理

#### ユーザー権限の付与

1. サイドメニューから「設定」→「権限管理」をクリック
2. 「権限を追加」ボタンをクリック
3. Larkユーザーを検索して選択
4. 権限（管理者/閲覧者）を選択
5. 「追加」ボタンをクリック

#### ユーザー権限の変更

1. 権限一覧から対象ユーザーを選択
2. 権限を変更
3. 「保存」ボタンをクリック

#### ユーザー権限の削除

1. 権限一覧から対象ユーザーの「削除」ボタンをクリック
2. 確認ダイアログで「削除」をクリック

### 12.2 システム設定

#### 会社情報（許可証PDF用）

| 設定項目 | 説明 |
|----------|------|
| 会社名 | 許可証に印字される会社名 |
| 郵便番号 | 会社の郵便番号 |
| 住所 | 会社の住所 |
| 発行部署 | 許可証の発行部署名 |

#### 有効期限警告設定

| 設定項目 | 説明 | デフォルト |
|----------|------|------------|
| 免許証警告日数 | 有効期限の何日前に警告するか | 30日 |
| 車検証警告日数 | 車検有効期限の何日前に警告するか | 30日 |
| 任意保険証警告日数 | 保障終了日の何日前に警告するか | 30日 |
| 管理者エスカレーション日数 | 期限切れ後何日で管理者に通知するか | 7日 |

## トラブルシューティング

### Q1. 承認ボタンが表示されない
A1. 閲覧者権限では承認できません。管理者に連絡してください。

### Q2. 社員が検索で見つからない
A2. Larkに登録されている社員のみ検索可能です。人事部に確認してください。

### Q3. 許可証が発行できない
A3. 以下を確認してください：
- 免許証が承認済みか
- 車検証が1件以上承認済みか
- 任意保険証が1件以上承認済みか

### Q4. 通知が届かない
A4. 以下を確認してください：
- 対象社員のLark IDが正しいか
- Larkの通知設定が有効か
- システム設定の警告日数が適切か
