# 車両関連書類管理システム - システム概要

## 1. システム概要

本システムは、社員の車両関連書類（運転免許証、車検証、任意保険証）を一元管理し、有効期限の監視と許可証の発行を行うWebアプリケーションです。

### 1.1 主な機能

| 機能 | 説明 |
|------|------|
| 書類申請 | 運転免許証・車検証・任意保険証の登録申請 |
| 書類承認 | 管理者による申請書類の審査・承認・却下 |
| 許可証発行 | 承認済み書類に基づく許可証PDFの発行 |
| 有効期限監視 | 書類の有効期限切れ警告と管理者エスカレーション |
| 代理申請 | 管理者・閲覧者による社員の代理申請 |

### 1.2 ユーザー権限

| 権限 | 対象者 | できること |
|------|--------|------------|
| 申請者 | 全社員 | 自分の書類申請、許可証確認 |
| 閲覧者 | 総務担当者等 | 書類閲覧、代理申請 |
| 管理者 | システム管理者 | 全機能（承認、設定変更含む） |

### 1.3 対応書類

1. **運転免許証** - 1人1件まで
2. **車検証** - 複数登録可能
3. **任意保険証** - 複数登録可能

## 2. システム構成

### 2.1 画面構成

```
申請者向け（/dashboard）
├── ダッシュボード（書類申請状況）
├── 免許証申請
├── 車検証申請
├── 任意保険証申請
├── マイ書類一覧
└── 許可証確認

管理者・閲覧者向け（/admin）
├── 申請一覧・承認
├── 社員別書類確認
├── 代理申請
├── 社員検索
├── 許可証発行
├── 有効期限監視
├── 通知履歴
└── 設定
    ├── 権限管理
    └── システム設定
```

### 2.2 技術構成

- **フロントエンド**: Next.js 15 (App Router)
- **認証**: Lark OAuth + NextAuth.js
- **データベース**: Lark Base
- **通知**: Lark Messenger

---

## 3. アクセス方法

### 3.1 ログイン

1. システムURLにアクセス
2. 「Larkでログイン」ボタンをクリック
3. Larkアカウントで認証

### 3.2 ログイン後の遷移

- **管理者・閲覧者** → 管理画面（/admin/applications）
- **一般社員** → ダッシュボード（/dashboard）
